# Clicker Example With React.js

This is my first react.js project. 

It's actually just a web page you can click on. The only difference is that you determine the duration of time you can click.

I used:
+ useEffect,
+ useRef,
+ useState,
+ tailwind.css,
+ css-animation-sync
